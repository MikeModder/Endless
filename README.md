# Endless

Endless is a Discord bot written in Java using [Java Discord API (JDA)](https://github.com/DV8FromTheWorld/JDA) and [JDA-Utilities](https://github.com/JDA-Applications/JDA-Utilities) with love ‚ù§.

## Editing the source code

I won't provide support when editing the source code to create a custom build of Endless (unless its going to be a [pull request](https://github.com/EndlessBot/Endless/pulls)).
